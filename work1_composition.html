Database Design for Unmanned Payment System by Using YOLO

  • 프로젝트 배경
  • 프로젝트 목적
  • 무인결제 시스템 구성
  • 영상 데이터 처리
  • 데이터베이스 구성 및 처리

무인 결제 시스템의 구성은 이미지와 같습니다. 매장 입장 시 RFID 혹은 스마트폰 근거리 무선 통신(NFC)기능을 활용하여
매장 내 고객을 식별합니다. 식별된 고객에 대하여 라즈베리파이의 카메라 모듈로 영상 데이터를 수집합니다.
그리고 이 데이터를 YOLO를 통해 분석하고, 분석된 영상 결과를 데이터베이스에 저장합니다.
데이터베이스에서는 매장 내 고객 식별, 상품 선택 여부, 최종 결제에 따른 데이터 저장이 이루어집니다.

YOLO를 선택한 이유는 다양한 상황에서 소요된 시간 대비 높은 정확도를 보여주며, CNN등 여러 신경망과 비교해
전체 이미지를 직접 학습해 실제 응용 프로그램에서 활용하기에 적합하기 때문입니다.
또한, 검출 성능에 직접적으로 대응하는 손실 함수에 대해 훈련하고, 그에 따라 처리시간 측면에서 실시간
객체 검출이 가능합니다. 다른 검출기에 비해 객체 표현을 보다 일반화하여 빠르고 강력한 검출이 가능하기에
객체 검출에서 보다 효율적이라고 판단하여 선택하게 되었습니다.
